# Job Application Tracker

A full-stack web application to help users manage and track their job applications efficiently.

## Features
- **Add, Edit, and Delete** job applications.
- **Search** by company name and **filter** by status (Applied, Interview, Offer, Rejected).
- **Dashboard** for a quick overview of application statuses.
- **Notes** section for interview preparation.

## Tech Stack
- **Frontend:** React
- **Backend:** Node.js with Express
- **Database:** MySQL
- **Styling:** Tailwind CSS
